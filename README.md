# brackets

Python script (parentheses_are_balanced.py) to checkup for balanced parentheses in an string literal.

Example:
```
parentheses_are_balanced("[{{}({})]")  # balanced
parentheses_are_balanced("[[")  # not balanced
parentheses_are_balanced("[{]}")  # not balanced
parentheses_are_balanced("{{([])}}")  # balanced
```
